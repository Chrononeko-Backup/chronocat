欢迎来到开发！Chronocat
在实现了多种协议的同时也提供了多种框架的接入，你可以根据自己的需求选择最适合的开发方式。

## 基于框架（推荐）

基于框架进行开发是 Chronocat 最推荐的开发方式。

要基于框架进行开发，请先将 Chronocat 接入框架，然后参考对应框架的开发文档。

  className="button button--secondary button--lg button--normal"
  to="/connect"
>
  <span>前往接入</span>
</Link>

## 基于协议

Chronocat 同时支持 Satori 和 Red 两种协议。我们推荐新用户基于 Satori 协议进行开发。

### Satori Protocol（推荐）

Satori Protocol 是 Chronocat 推荐使用的协议实现。
