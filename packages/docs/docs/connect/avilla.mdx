[Avilla](https://graia.readthedocs.io/avilla) 是 [Graia Project](https://graia.readthedocs.io) 的 "下一代" 框架实现，
通过对 [ `OneBot` ](https://github.com/botuniverse/onebot)、[ `Telegram Bot API` ](https://core.telegram.org/bots)
及其他的基于不同即时通讯软件实现的各式对接接口，
以及其他具有相应概念的“异步消息流”实例进行总结，抽象其中最为基本的各式模型，构造了一个理论上可以实现零成本切换对接平台的框架。

将 Chronocat 接入 Avilla 可以查看 avilla-satori 项目。
